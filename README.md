# React QR Genius

## Description

> A simple React application that allows users to generate, download, and print QR codes. Built with React and Bootstrap for a sleek and responsive interface.

## Features

- Generate QR Codes: Enter any text or URL to generate a QR code instantly.
- Download QR Codes: Download the generated QR code as a PNG image.
- Print QR Codes: Print the QR code directly from the app.

## Get Started

### Run

```
To get a clone of the project, run `https://github.com/odhiambo-ed/react-qr-genius.git`
```

```
cd /react-qr-genius
```

```
npm install
```

```
npm run dev
```
